# 🚀 Atomize – Goal Breakdown App

Atomize is a mobile productivity app built with **Flutter**  and **Node** that helps users turn big goals into simple daily tasks and track their progress consistently.

---

## 📱 Features

* 🔐 User Authentication (Signup & Login)
* 🎯 Create and manage goals
* ⚙️ Automatically break goals into daily tasks
* 📋 Track and update tasks
* 📊 Visualize progress and growth

---

## 🎨 Frontend (Flutter)

* Built with Flutter (Dart)
* Clean and modern UI design
* Smooth onboarding experience
* Goal tracking dashboard
* Task management screens
* Progress visualization

---

## 🔌 Backend Integration

* REST API built with Node.js & Express
* JWT-based authentication
* MongoDB database

---

## ⚙️ Setup (Frontend)

```bash
git clone https://github.com/your-username/atomize.git
cd atomize
flutter pub get
flutter run
```

---

## 🔑 API Usage

* Base URL: `http://localhost:5000/api`
* Auth endpoints:

  * `POST /auth/signup`
  * `POST /auth/login`
